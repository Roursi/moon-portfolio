## 个人介绍单页（无框架）

### 本地预览
- 直接双击打开 `index.html` 也可以预览
- 更推荐本地服务（避免部分浏览器限制资源加载）：

```bash
cd /Users/mac/Downloads/personal_page
python3 -m http.server 8000
```

然后打开 `http://localhost:8000`

### 期刊 PDF 放置位置（5 个）

把你的 5 个 PDF 放到下面路径，并按命名规则命名：

- `assets/journals/journal-1.pdf`
- `assets/journals/journal-2.pdf`
- `assets/journals/journal-3.pdf`
- `assets/journals/journal-4.pdf`
- `assets/journals/journal-5.pdf`

页面里点击“期刊设计”卡片会弹窗打开 PDF，并支持翻页（上一页/下一页，键盘方向键也可翻页）。
